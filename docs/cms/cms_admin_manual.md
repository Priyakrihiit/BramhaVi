# CMS Admin Manual

## 1. Supervisor Capabilities

Administrators have access to global management components under the **Enterprise CMS Hub**:

- **Workflow Review**: Review incoming review states under the **Workflow** tab. Approve or request changes.
- **Comment Moderation**: Visit the **Moderation** tab. Click **Moderate** on flagged inputs to overwrite contents with a warning banner.
- **URL Redirects**: Configure path redirects under the **Settings/Redirects** tab to map legacy URLs.
- **Revisions Rollback**: Access the **Revisions** tab to rollback content states to previous snapshots.
- **System Telemetry**: Inspect traffic counters, content tallies, and audit logs.
