# User Manual: Enterprise Analytics Platform

**BrahmaVidya Galaxy — Analytics Module**
**Version:** 1.0 | **Audience:** Students, Learners

---

## 1. Overview

The Analytics Dashboard gives you insight into your personal learning journey — study time, course completions, badges earned, and certificates issued. It is always available in the platform's Admin Control Center.

---

## 2. Accessing Your Dashboard

1. Log in to the BrahmaVidya Portal.
2. Click **Control Center** from the top navigation.
3. Select **Enterprise Analytics** from the left sidebar.
4. The **Control Dashboard** tab loads automatically.

---

## 3. Control Dashboard Tab

The Control Dashboard shows your key metrics at a glance:

| Widget | Description |
|---|---|
| **Lessons Started** | Total lessons you have opened this month |
| **Courses Completed** | Number of courses finished |
| **Certificates Issued** | Verified completion credentials earned |
| **Badges Awarded** | Gamification milestone achievements |
| **Practice Sessions** | Number of practice question sets attempted |

Each widget card displays the current value. Values update automatically every 10 seconds.

---

## 4. Module Analytics Tab

Click **Module Analytics** to explore breakdown charts per domain:

- **User** — Login history and activity patterns
- **Course** — Lesson progression timeseries
- **Search** — Topics you have searched for
- **Notifications** — Notification delivery rate

Use the **module selector** buttons at the top to switch between domains.

The **Weekly Performance Timeseries** chart plots your activity across the last 7 days.

---

## 5. Realtime Telemetry Tab

Shows live platform activity at the moment:
- **API Gateway Request Rate** — Real-time line chart of server requests per second.
- **Geographic Heatmap** — Country-level user density breakdown.

---

## 6. Frequently Asked Questions

**Q: How often is the dashboard updated?**
A: Summary statistics are computed nightly. Realtime counters update every 10 seconds.

**Q: Can I download my data?**
A: Contact your institution administrator to request a personal activity export report.
