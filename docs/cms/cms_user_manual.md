# CMS User Manual

## 1. Accessing the Platform
1. Log in to BrahmaVidya Academy.
2. If you are registered as a Creator, Teacher, or Supervisor, click **Enterprise CMS Hub** in the sidebar.

## 2. Managing Articles & Content
- **Drafting**: Click **Create Article** on the Articles tab. Input title, slug, and markdown content. Click **Save Changes**.
- **Publishing**: Click the Globe icon to publish immediately.
- **Previewing**: Click the preview edit fields to verify layout appearances prior to making articles live.
- **Uploading Media**: Go to the **Media Library** tab. Click **Upload Media Asset** to store files. Copy URL nodes directly into draft bodies.
