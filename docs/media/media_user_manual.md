# Enterprise DAM User Manual

## 1. Accessing the Platform
1. Log in to BrahmaVidya Academy.
2. Click **Enterprise CMS Hub** in the sidebar.
3. Open the **Media Library** tab.

## 2. Managing Folders & Collections
- **Directories Hierarchy**: Add folders using the sidebar text inputs to structure files by categories.
- **Upload Assets**: Upload photos/videos directly to folders.
- **Inspect Metadata**: Click on files to view dimensions, formats, and share paths.
