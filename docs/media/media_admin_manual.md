# Enterprise DAM Admin Manual

## 1. Supervisor Capabilities
Administrators have access to global management components under the **Enterprise CMS Hub**:
- **Workflow Review**: View pending uploads and approve/reject assets before they are marked public.
- **Recycle Bin**: Access deleted files, restore files, or permanently delete items.
- **Audit Trails**: Inspect before/after state snapshots.
- **Storage Settings**: Set max file limits and scan configurations.
